Download the test questions
Click "Download Test Questions" button and save the text file to a location on your computer hard drive (eg. downloads folder or Desktop).
 
 Navigate to the test section in Blackboard
  1. In Blackboard, Click on "Control Panel" - "Course Tools
  2. Click "Test, Surveys and Pools"
  3. Click on "Tests"
Open the test canvas
  1. Click on the Chevron beside the test you wish to add the questions to.  You may have to Click on "Build Test" if the test dosn't exits yet.
  2. Click "Edit"
  1. Click on "Upload Questions"  button
Select the questions test file to add
  1. Browse to the text file from the Quiz generator
  2. Enter the point value per question and click on the submit button
